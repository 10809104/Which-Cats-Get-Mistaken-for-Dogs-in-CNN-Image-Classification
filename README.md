# 哪種貓會被誤辨識為狗 (Which Cats Get Mistaken for Dogs in CNN Image Classification)

## 專案概覽 (Project Overview)
這個專案旨在探索哪些貓品種最容易在CNN圖像分類中被誤辨識為狗。透過訓練一個貓狗分類模型，我們將分析哪些貓品種最容易被混淆成狗。

This project aims to explore which cat breeds are most commonly misclassified as dogs by Convolutional Neural Networks (CNNs) in image classification tasks. By training a model to classify cats and dogs, we will investigate which cat breeds are most likely to be confused with dogs.

## 研究問題 (Research Questions)
- 哪些貓品種最容易被誤辨識為狗？
- 什麼樣的特徵（例如：大小、毛皮類型、臉部特徵）會導致誤分類？

Which cat breeds are most likely to be misclassified as dogs?
What characteristics (e.g., size, fur type, facial features) contribute to misclassification?

## 方法論 (Methodology)
- 資料集 (Dataset): 我們將使用Oxford Pets資料集或其他包含多種貓狗品種的資料集。
- 模型 (Model): 使用CNN模型（例如ResNet或VGG）進行貓狗分類。
- 分析 (Analysis): 分析誤分類案例，並確定哪些貓品種最容易被誤分類為狗。

Dataset: We will use the Oxford Pets dataset or other similar datasets containing a variety of cat and dog breeds.
Model: A CNN model (e.g., ResNet or VGG) will be trained to classify images as cats or dogs.
Analysis: We will analyze misclassified instances and identify which cat breeds are confused with dogs.

## 未來工作 (Future Work)

- 研究某些特徵（例如毛皮花紋、耳形等）如何導致貓被誤分類為狗。

In-depth analysis of misclassified cases.
Investigating why certain features (e.g., fur patterns, ear shapes) cause misclassification.
